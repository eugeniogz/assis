Bem-vindo ao Login UFMG

Você está sendo redirecionado para a página de login dos Sistemas UFMG.

Se o redirecionamento não ocorrer automaticamente, clique aqui para acessar.

Para instalar este aplicativo na sua tela inicial:

1. Aguarde esta página carregar completamente.

2. No seu navegador (Chrome, Edge, etc.), procure pelo botão de "Instalar" (geralmente um ícone de casa com um sinal de mais, ou uma opção no menu).

3. Toque em "Adicionar à tela inicial" ou "Instalar aplicativo".

Assim, você terá um ícone direto para o login da UFMG!
