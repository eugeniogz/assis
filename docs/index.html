Doctoral Thesis

Tese de Doutorado

Agile Method for Semantic Integration of Scientific Data Based on Ontologies Método Ágil de Integração Semântica de Dados Científicos Baseado em Ontologias

Developed the agile method "Odin" for semantic data integration to ingest heterogeneous scientific data from sensors, smartbands, and epidemiological databases into RDF knowledge graphs.

Desenvolvimento do método ágil "Odin" de integração semântica de dados para ingerir dados científicos heterogêneos de sensores, smartbands e bases epidemiológicas em grafos de conhecimento RDF.

Advisor: Prof. Dr. Marcello Peixoto Bax Orientador: Prof. Dr. Marcello Peixoto Bax 2020
