Doctoral Thesis

Tese de Doutorado

Agile Method for Semantic Integration of Scientific Data Based on Ontologies Método Ágil de Integração Semântica de Dados Científicos Baseado em Ontologias

Developed the agile method "Odin" for semantic data integration to ingest heterogeneous scientific data from sensors, smartbands, and epidemiological databases into RDF knowledge graphs.

Desenvolvimento do método ágil "Odin" de integração semântica de dados para ingerir dados científicos heterogêneos de sensores, smartbands e bases epidemiológicas em grafos de conhecimento RDF.

Advisor: Prof. Dr. Marcello Peixoto Bax Orientador: Prof. Dr. Marcello Peixoto Bax 2020

Performance Comparison: Linux × Windows NT Comparação de desempenho Linux × Windows NT

A comparative methodology evaluated on identical hardware, measuring wall-clock times under synthetic workloads, multiprogrammed network clients (TCP/IP and HTTP), and Linpack benchmarks.

Metodologia comparativa avaliada em hardware idêntico, medindo tempos de execução sob cargas de trabalho sintéticas, clientes de rede multiprogramados (TCP/IP e HTTP) e benchmarks Linpack.

Doctoral Research Artifacts

Recursos da Tese de Doutorado

Artifacts and semantic mappings generated for data ingestion using HADatAc V 1.1.12, an ontology-driven platform for semantic integration of scientific data.

Artefatos e mapeamentos semânticos gerados para ingestão de dados utilizando HADatAc V 1.1.12, uma plataforma orientada a ontologias para integração de dados científicos.

  • Smartbands Data Templates

    Modelos de Dados de Smartbands

    Metadata templates for ingestion of Fitbit and Mi Band activity and physiological logs.

    Modelos de metadados para ingestão de registros de atividade física e dados fisiológicos das pulseiras Fitbit e Mi Band.

    demo_templates demo-ont.ttl (Ontology)

  • Main Experiment & Epidemiological Mappings

    Experimento Principal & Mapeamentos Epidemiológicos

    Ingestion templates and ontologies representing Dengue and Schistosomiasis data in Brazil integrated with environmental variables (temperature, precipitation) and socioeconomic metrics.

    Templates de ingestão e ontologias representando dados de Dengue e Esquistossomose no Brasil integrados a variáveis ambientais (temperatura, precipitação) e métricas socioeconômicas.

    demo_gbd base.ttl (Base Ontology) base graph
